# **App Name**: ShipShape

## Core Features:

- Hero Section: Display a clear and concise landing page showcasing the core value proposition.
- Call to Action Buttons: Provide an exporter signup button/link and a carrier portal button/link.
- How it Works Section: Provide a brief explanation of how Shipment Battlefield works.
- Authentication UI: Implement Login and Signup user interfaces

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a sense of trust and efficiency, reflecting the logistical nature of the platform.
- Background color: Very pale blue (#F0F8FF) to create a clean and open feel.
- Accent color: Pale greenish-blue (#7DF9FF), providing a contrast to the primary while maintaining a cohesive, modern look.
- Font pairing: 'Poppins' (sans-serif) for headers, and 'PT Sans' (sans-serif) for body text.
- Use simple, professional icons to represent key features and benefits.
- Maintain a clean, spacious layout with clear sections for easy navigation.
- Subtle animations for button interactions to add a touch of interactivity.
